In a first terminal, launch :

roslaunch vision_processing green_cube_feeding_casf.launch

ou 

roslaunch vision_processing green_cube_feeding_casf.launch       cbf_node_type:=cbf_safety_node_Bernstein_multicbf.py       cbf_direct_position_mode:=integrated

roslaunch vision_processing green_cube_feeding_casf.launch     cbf_node_type:=cbf_safety_node_Bernstein_multicbf.py     cbf_direct_position_mode:=integrated     cbf_solver_mode:=fast_tangent     cbf_recovery_speed:=0.02     cbf_recovery_max_correction_speed:=0.55

Pour sans CBF :

roslaunch vision_processing green_cube_feeding_casf.launch       cbf_node_type:=cbf_safety_node_Bernstein_multicbf.py       cbf_direct_position_mode:=integrated       cbf_monitor_only:=true       contact_stop_enabled:=true       contact_stop_clearance:=0.0005

In a second terminal, launch :

rosbag record -O run1.bag /cbf_safety/diagnostics /cbf_safety/diagnostics_layout

pour les trajectoires, ajouté :

rosbag record --lz4 -O run21.bag /cbf_safety/diagnostics /cbf_safety/diagnostics_layout /joint_states /perception/persistent_obstacles
rosbag record --lz4 -O runPP1.bag -e \
    '/cbf_safety/diagnostics(_layout)?$|/joint_states$|/perception/persistent_obstacles$|/pipeline/timing/.*'


rosbag record --lz4 -O runPP3.bag -e '/cbf_safety/diagnostics(_layout)?$|/joint_states$|/planner/nominal_joint_velocity$|/planner/
  nominal_joint_command$|/planner/nominal_trajectory$|/joint_group_velocity_controller/command$|/perception/persistent_obstacles$|/perception/
  cleaned_obstacles$|/pipeline/timing/.*'
  
When it is done, run :

python3 scripts/plot_cbf_diagnostics.py runPP1.bag --csv for the plots of one bag

For comparison : 

python3 scripts/analyze_trajectory.py run7.bag run8_NO_CBF.bag -o trajectory_comparison_3
